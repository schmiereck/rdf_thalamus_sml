Based on the project documents provided, this is a **bootstrap state** — no experiments have been run yet, and Phase 0 (Harness & Smoke Test) is the first task. Let me formulate the plan for Phase 0.

## Analysis

**Current State:** Empty codebase, no iterations completed. The project needs its foundational experimental harness and the five Phase 0 encoder implementations.

**Phase 0 Purpose:** This is explicitly a smoke test, not a research result. The goal is to:
1. Build the experimental pipeline (dataset generation, similarity metrics, evaluation)
2. Implement all five encoder alternatives (P0-A through P0-E)
3. Verify the similarity metric works (Spearman ρ ≥ 0.6 for non-trivial methods)
4. Produce `phase_0/REPORT.md`

**Key Design Decisions for the Harness:**
- **Input space:** 3 binary pixels → 8 possible states, enumerated with noise variants
- **Similarity metric:** Cosine similarity of codes vs inverse Hamming distance of inputs, measured via Spearman rank correlation
- **Local training constraint:** P0-B (SDR), P0-C (SOM), P0-E (Predictive Coding) must train via local rules only; P0-D (Sparse AE) uses reconstruction loss; P0-A is trivial
- **Dimension d:** Must be chosen so input slot dim = output slot dim for recursability. With 3 binary inputs, a reasonable starting d is 8–16 (enough to represent 8 states sparsely)

**Hypothesis Rationale:** Since Phase 0 has only 8 input states, every reasonable method should succeed. The interesting claim is that **locally-trained encoders can match globally-trained ones** at this trivial scale. If they can't even match at this scale, the entire local-learning premise is in trouble. The 0.15 gap threshold is generous — if local methods are more than 15% worse than a global optimizer even with 8 states, that's a warning sign.
