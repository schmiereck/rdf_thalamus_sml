### CRITICAL CONTEXT from Diagnostic (iter 6.2)

Two root causes identified for Phase 3 failure:

**Problem 1: VICReg is INEFFECTIVE on intermediate codes**
- Per-dim std at all layers: ~0.07-0.08 (VICReg target: ≥1.0)
- Variance DECREASED during training (not increased)
- Math: VICReg gradient per element = λ/(M·d·std). For M=B·T·S=28672 (spatial layer 0), gradient ≈ 0.0008/elem. Prediction gradient ≈ 0.016/elem. VICReg is overwhelmed ~20×.
- FIX: Add a separate VICReg loss on the POOLED representation (shape (B, d_out) = (64, 16)). Here M=B=64, so gradient per element ≈ 25/(64·16·0.07) ≈ 0.33/elem — 400× stronger than per-layer VICReg.

**Problem 2: Mean-pooling destroys discriminative information**
- Pooled (B,16): 44.25% test acc
- spatial_pooled_then_flat (B,416): 53.5% test acc  (+9.25pp)
- FIX: Use spatial_pooled_then_flat as the classification readout. This means: from the final temporal output (B, 26, 10, 16), mean-pool over spatial (axis=2) → (B, 26, 16), then flatten → (B, 416).

1. **Add pooled VICReg function** (implement in the new file):
```python
def pooled_vicreg_loss(z_pooled, eps=1.0):
    """
    Compute VICReg variance + covariance loss on pooled representation.
    z_pooled: (B, d) 
    Returns: var_loss, cov_loss (scalars)
    """
    B, d = z_pooled.shape
    # Variance: mean of max(0, 1 - std_j) over dimensions
    std = np.sqrt(z_pooled.var(axis=0, ddof=0) + 1e-12)
    var_loss = float(np.mean(np.maximum(0.0, eps - std)))
    
    # Covariance: (1/d) * sum of off-diagonal C_{j,k}^2
    zc = z_pooled - z_pooled.mean(axis=0, keepdims=True)
    cov = (zc.T @ zc) / B
    mask = 1.0 - np.eye(d)
    cov_loss = float(np.sum(cov ** 2 * mask) / d)
    
    return var_loss, cov_loss
```

2. **Add pooled VICReg gradient function**:
```python
def pooled_vicreg_grad(z_pooled, lambda_var=25.0, lambda_cov=25.0, eps=1.0):
    """
    Compute gradient of VICReg loss w.r.t. z_pooled.
    z_pooled: (B, d)
    Returns: dL_dz_pooled: (B, d)
    """
    B, d = z_pooled.shape
    mu = z_pooled.mean(axis=0, keepdims=True)
    std = np.sqrt(z_pooled.var(axis=0, ddof=0) + 1e-12)
    
    # Variance gradient
    mask = (std < eps).astype(float)
    dz_var = -(lambda_var / d) * mask * (z_pooled - mu) / (B * std + 1e-12)
    
    # Covariance gradient
    zc = z_pooled - mu
    cov = (zc.T @ zc) / B
    C_off = cov * (1.0 - np.eye(d))
    dz_cov = (lambda_cov * 4.0 / (B * d)) * (zc @ C_off)
    
    return dz_var + dz_cov
```

3. **Modify train_jepa_epoch** to optionally include pooled VICReg:
   - After the forward pass, compute pooled VICReg loss on `fwd["pooled"]`
   - Compute pooled VICReg gradient
   - The pooled VICReg gradient needs to be backpropagated through the encoder. This is tricky because the current backward() expects code-level gradients, not pooled-level gradients.
   
   **Simpler approach**: Since the pooled representation is `x_temporal.mean(axis=(1,2))`, the gradient of the loss w.r.t. x_temporal is `dL_dz_pooled` expanded back: `dL_dx_temporal = dL_dz_pooled[:, None, None, :] / (T_final * S_final)` where T_final=26, S_final=10. This is then added to the temporal code gradients.

   Actually, the simplest approach is to add the pooled VICReg gradient as an additional term in `dL_dtemporal_codes[-1]`:
   ```python
   # After JEPA training step, add pooled VICReg gradient
   if use_pooled_vicreg:
       dL_dpooled = pooled_vicreg_grad(fwd["pooled"])
       # Expand to temporal output shape: (B, T_final, S_final, d)
       # pooled = x_temporal.mean(axis=(1,2))
       # So dL/dx_temporal = dL/dpooled / (T_final * S_final) broadcast
       dL_dpooled_expanded = dL_dpooled[:, None, None, :] / (T_final * S_final)
       dL_dtemporal_codes[-1] += dL_dpooled_expanded  # Add to last temporal layer's gradient
   ```

   Wait, this isn't quite right. The pooled VICReg gradient should flow through the entire backward pass, not just the last temporal layer. But adding it to the last temporal layer's code gradient is the most direct path.

   Actually, the encoder's backward() function takes `dL_dtemporal_codes` which are gradients w.r.t. each temporal layer's output. The gradient of the pooled representation w.r.t. the last temporal layer's output is: `dL/dpooled * dpooled/dx_temporal = dL/dpooled / (T*S)`. This is correct.

   But we also need to consider: the pooled VICReg gradient should also flow back through the spatial layers (since the temporal layers depend on spatial outputs). This happens naturally through the backward() function — the temporal code gradient propagates backward through the temporal layers and then through the spatial layers.

   ```python
   def evaluate_classification_v2(encoder, train_grid, train_y, test_grid, test_y, 
                                  seed=42, readout='spatial_pooled_then_flat'):
       fwd_train = encoder.forward_with_intermediates(train_grid)
       fwd_test = encoder.forward_with_intermediates(test_grid)
       
       if readout == 'pooled':
           train_repr = fwd_train["pooled"]
           test_repr = fwd_test["pooled"]
       elif readout == 'spatial_pooled_then_flat':
           # temporal_outputs[-1] shape: (B, T_final=26, S_final=10, d=16)
           # Mean over spatial (axis=2): (B, 26, 16)
           train_repr = fwd_train["temporal_outputs"][-1].mean(axis=2).reshape(len(train_y), -1)
           test_repr = fwd_test["temporal_outputs"][-1].mean(axis=2).reshape(len(test_y), -1)
       
       probe = SimpleLogisticRegression(n_classes=N_CLASSES, n_features=train_repr.shape[1], 
                                         lr=0.1, max_iter=2000, seed=seed)
       probe.fit(train_repr, train_y)
       train_acc = float(probe.score(train_repr, train_y))
       test_acc = float(probe.score(test_repr, test_y))
       return {"train_acc": train_acc, "test_acc": test_acc}
   ```
